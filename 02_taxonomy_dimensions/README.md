# Paper 2: The DNA of Business Meetings: A Dimensional Framework

## Project Focus
This paper shifts from technology to task analysis. It aims to decode business meetings into atomic, objective dimensions (e.g., physical vs. abstract, structured vs. creative) to build a universal taxonomy. This framework will algorithmically determine if VR offers tangible benefits for specific meeting configurations.

## Local Data Dependencies
* `Dimensionen.csv`, `Meeting Types.csv`, `Arbeitstabelle Meeting Types.csv`, `Sources Taxonomy.csv`.

## Sprints / Work Packages
### Sprint 1: Literature & Taxonomy Review
* Analyze existing grey literature and academic taxonomies to identify gaps in current meeting classifications.

### Sprint 2: Framework Construction
* Synthesize the `Dimensionen.csv` data to finalize the comprehensive dimensional framework for professional meetings.

### Sprint 3: Meeting Type Mapping
* Map the identified dimensions onto standard meeting types (from `Meeting Types.csv`) to create a structured database of meeting requirements.

### Sprint 4: VR Impact Hypothesis Formulation
* Systematically evaluate how VR's unique properties (e.g., 6DOF, spatial audio, virtual presence) impact each specific dimension.

### Sprint 5: Manuscript Drafting
* Write the scientific article detailing the development of the taxonomy, the dimensional framework, and its application for assessing technological suitability.
